1.1 Character Set in C++

Character Set refers to the set of valid characters that are allowed in a C++ program.
These characters are used to form keywords, identifiers, constants, operators, and symbols.

Types of Characters in C++

1. Letters
- Uppercase letters: A to Z
- Lowercase letters: a to z

2. Digits
- 0 to 9

3. Special Characters
- +  -  *  /  %
- =  <  >  !
- &  |  ^  ~
- #  @  $
- ( )  { }  [ ]
- ;  :  ,  .

4. White Spaces
- Space
- Tab
- New line

Use of Character Set
- Used to write keywords and identifiers
- Used in expressions and statements
- Used to define operators and symbols

Note:
C++ also supports extended character sets such as Unicode, but basic programs mostly use ASCII characters.
